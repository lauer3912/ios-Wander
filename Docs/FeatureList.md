# Wander — Feature List

## Overview
- **App Name**: Wander
- **Bundle ID**: com.ggsheng.Wander
- **Concept**: AI-generated fictional strangers for anonymous emotional conversation
- **Target**:欧美青少年/青年 (15-35岁)，需要情感陪伴但不想暴露身份

---

## Feature Categories

### 1. Onboarding & First Run
1. Animated intro sequence (3 screens)
2. Personality quiz (5 questions to determine preferred stranger archetypes)
3. Anonymous nickname setup
4. Privacy introduction screen
5. Notification permission request
6. Theme preference (Dark default, Light option)

### 2. Stranger System
7. Random stranger assignment algorithm
8. Stranger character card (name, archetype, mood, backstory)
9. 12+ unique character archetypes (Wanderer, Lost Poet, Night Owl, etc.)
10. Character mood indicators (Excited, Thoughtful, Melancholy, Playful)
11. "Meet New Stranger" button (cooldown timer 24h)
12. Stranger conversation history per character
13. Favorite strangers list
14. Block/report stranger (with reason selection)
15. Character backstory generation (on first meet)
16. Stranger memory (remembers your chat tone across sessions)

### 3. Chat System
17. Real-time message display
18. AI-generated stranger responses (contextual, emotionally aware)
19. Typing indicator animation
20. Message timestamp display
21. Message bubble styling (user vs stranger differentiated)
22. Voice-to-text input option
23. Emoji picker
24. Message copy action
25. Long-press to report message
26. Conversation end screen (rate feeling before/after)
27. "Share thoughts" prompt after conversation

### 4. Emotional Check-In
28. Pre-chat mood selector (5 moods: Anxious, Sad, Lonely, Neutral, Hopeful)
29. Post-chat mood selector (same 5 + "Better", "Same", "Worse")
30. Mood trend chart (weekly view)
31. "How are you really feeling?" daily prompt
32. AI sentiment analysis of conversation tone
33. Mood improvement notification (when streak active)

### 5. Journal (Private)
34. AI-guided reflection prompts after chat
35. Personal notes (unshared, fully private)
36. Date/time/location tag on entries
37. Mood vs journal correlation chart
38. Journal search by keyword
39. Export journal as text (local only)
40. Journal streak counter

### 6. Gamification & Rewards
41. XP earned per conversation (10-50 XP based on length)
42. Level system (1-50, unlocking rewards)
43. Daily conversation streak (with flame icon)
44. Achievement badges (First Chat, 7-Day Streak, Mood Lifter, etc.)
45. Weekly progress report notification
46. Unlock new character archetypes at levels
47. Anonymous leaderboard (optional opt-in)

### 7. Profile & Stats
48. Anonymous profile display (nickname + avatar)
49. Total conversations count
50. Total time spent chatting
51. Favorite archetype chart
52. Mood trend line graph (30-day)
53. Achievement showcase (3 featured badges)
54. "Share my journey" (generates anonymous stats card for social)

### 8. Settings
55. Edit nickname
56. Change avatar (select from preset silhouettes)
57. Notification preferences (daily prompt, streak reminder, etc.)
58. Privacy mode (ghost mode — no logs)
59. Clear all data (with confirmation)
60. Block contacts (prevent matching same person)
61. Dark/Light theme toggle
62. AI response style preference (Brief, Balanced, Deep)
63. Delete account (full data wipe)

### 9. Discover (Explore Tab)
64. Browse character archetypes before matching
65. Preview character personalities
66. "Request specific archetype" wishlist
67. Seasonal limited-time characters

### 10. Mini-Game (Wind Down)
68. Breathing exercise animation (4-7-8 technique)
69. "Thought dump" — rapid free-write timer (2 min)
70. Constellation maker (decorative, playful)
